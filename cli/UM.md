# CLI for Update Manager
Update Manager is similar in function to YUM & APT.  It is consistent across platforms & also works in sandbox mode.

## Synopsis
    ./pgedge um <command> [parameters] [options] 

[**list**](help/um-list.md) - Display available/installed components<br>
[**update**](help/um-update.md)  - Retrieve new list of components & update ctl<br>
[**install**](help/um-install.md) - Install a component (eg pg16, spock32, postgis, ...)<br>
[**remove**](help/um-remove.md) - Un-install component<br>
[**upgrade**](help/um-upgrade.md) - Perform an upgrade of a component<br>
[**clean**](help/um-clean.md) - Delete downloaded component files from local cache<br>
