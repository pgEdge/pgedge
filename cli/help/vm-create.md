## SYNOPSIS
    ./pgedge vm create PROVIDER AIRPORT NAME <flags>
 
## DESCRIPTION
    Create a virtual machine (VM)
 
## POSITIONAL ARGUMENTS
    PROVIDER
    AIRPORT
    NAME
 
## FLAGS
    --size=SIZE
    
    
    -i, --image=IMAGE
    
    
    --ssh_key=SSH_KEY
    
    
    -p, --project=PROJECT
