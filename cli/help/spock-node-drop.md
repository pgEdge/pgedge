## SYNOPSIS
    ./pgedge spock node-drop NODE_NAME DB <flags>
 
## DESCRIPTION
    Drop a spock node. 

Example: spock node-drop n1 demo
 
## POSITIONAL ARGUMENTS
    NODE_NAME
        The name of the node. Example: n1
    DB
        The name of the database. Example: demo
 
## FLAGS
    -p, --pg=PG
