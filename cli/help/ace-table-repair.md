## SYNOPSIS
    ./pgedge ace table-repair CLUSTER_NAME DIFF_FILE SOURCE_OF_TRUTH TABLE_NAME <flags>
 
## DESCRIPTION
    Apply changes from a table-diff source of truth to destination table
 
## POSITIONAL ARGUMENTS
    CLUSTER_NAME
    DIFF_FILE
    SOURCE_OF_TRUTH
    TABLE_NAME
 
## FLAGS
    -d, --dry_run=DRY_RUN
