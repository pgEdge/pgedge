## SYNOPSIS
    ./pgedge db create <flags>
 
## DESCRIPTION
    Usage:
    To create a database owned by a specific user
       db create -d <db> -U <usr> -P <passwd>
 
## FLAGS
    -d, --db=DB
    
    
    -U, --User=USER
    
    
    -P, --Passwd=PASSWD
    
    
    -p, --pg=PG
    
    
    -s, --spock=SPOCK
